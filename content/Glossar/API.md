# API

## Definition
Application Programming Interface (API) ist eine definierte Schnittstelle über die Software-Komponenten miteinander kommunizieren. APIs abstrahieren Implementierungsdetails und stellen Funktionen bereit.

## Typen

| Typ | Beschreibung |
|---|---|
| REST API | HTTP-basiert, JSON, Standard für Web |
| GraphQL | Flexiblere Abfragen, ein Endpoint |
| RPC/gRPC | Direkte Funktionsaufrufe, effizient |
| WebSocket | Bidirektional, Real-Time |
| SDK/Library API | Klassen und Methoden in Code |
