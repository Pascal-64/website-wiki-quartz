# Agent

## Definition
Im KI-Kontext: Ein Agent ist ein autonomes System das Ziele verfolgt, Aktionen plant und ausführt, dabei Werkzeuge (Tools) nutzt und auf Feedback reagiert. Basis ist meist ein [[LLM]] als Denkinstanz.

## Abgrenzung
- **Chatbot**: Antwortet auf Eingaben, führt keine eigenständigen Aktionen aus
- **Agent**: Plant, handelt, nutzt Tools, verfolgt Ziele über mehrere Schritte

## Eigenschaften
- Zielorientiert (Goal-directed)
- Verwendet Tools (Web-Suche, Code-Ausführung, APIs)
- Hat Gedächtnis (Context + Optional: Long-Term Memory)
- Kann Fehler korrigieren und iterieren
